# insert Function 
Add the new values in the array at specific position
# Syntax
```
np.insert(array,index,value,axis=None)
```
# Append
Add the new values at the end of the array
# Syntax
```
np.append(array,[array])
```
# Concatinate 
Combine the two Arrays along axis
# Syntax
```
np.concatinate((array1,array2), axis=0)
```
# Removing Elements of Array
Romove the element from the array
# Syntax
```
np.delete(array,index.no,axis=None)
```
# Stacking Arrays
Stack using vertically and horizontally
#### Function
- vstack() --> Row wise / vertical
- hstack() --> Column wise / Horizontal 
# Splitting Arrays
means divide the main Array into multiple sub-Array parts
#### Function
- np.split() --> used to divide array in equal parts
- np.hsplit() --> used to split array horizontally
- np.vsplit() --> used to split array vertically
